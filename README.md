# 🍳 Proyecto Cocina

Welcome to **Proyecto Cocina**! A comprehensive tool developed in Java for managing recipes, weekly menus, and shopping list optimization. Designed to organize your kitchen efficiently and professionally.

---

## 🚀 For Users

1. Go to the **[Releases](https://github.com/YutaKoyanagi10/ProyectoCocina/releases)** section.
2. Download the latest `ProyectoCocina-Installer.exe`.
3. Run the installer and follow the setup steps.
4. **exec as Administrator**Important!!

---

## 🛠 For Developers

### Tech Stack

* **Language:** Java 21
* **UI Framework:** JavaFX
* **Dependency Management:** Gradle

### Prerequisites

* **JDK 21** must be installed on your system.

### Environment Setup

1. **Clone the repository:**

   ```bash
   git clone [https://github.com/YutaKoyanagi10/ProyectoCocina.git](https://github.com/YutaKoyanagi10/ProyectoCocina.git)
   cd ProyectoCocina

2. Run the application

    ```bash
    ./gradlew run
    
3. Run test

    ```bash
    ./gradlew test
